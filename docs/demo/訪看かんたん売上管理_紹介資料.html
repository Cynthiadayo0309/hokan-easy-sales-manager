MONTHLY INPUT

人数を入れると、売上見込みを自動計算

対象年月と施設を選び、4つの看護区分の月次人数を入力。保存状態も日本語で分かりやすく表示します。

1施設タブで切り替え

施設ごとの「未入力・一時保存・入力完了」を確認しながら進められます。

2入力は0以上の整数

介護・別表7・精神・特指示の人数を、分かりやすい1列で入力します。

3その場で自動集計

月次人数合計と売上見込みが入力直後に更新されます。

4連続入力を支援

「保存して次の施設へ」で、5施設を順番に入力できます。

DATA & SAFETY

オフライン運用と、データ保全を両立

インターネット接続やクラウドアカウントを必要とせず、入力データは利用端末内に保存。出力とバックアップも画面から行えます。

✓

業務データは
このパソコンの中に

ローカルデスクトップアプリとして動作し、データベースはWindowsユーザーごとのアプリデータ領域へ保存されます。

PC
アプリを操作
→
DB
端末内に保存

出CSV出力

対象月の詳細CSVと月間集計CSVを保存。Excelで開きやすいUTF-8形式です。

保バックアップ

データベースをファイルとして保存。定期的な保全や端末移行に利用できます。

復安全な復元

復元は2段階確認。実行前に現在のデータを自動退避し、失敗時は現データを維持します。

個患者個人情報を扱わない

患者氏名・住所・病名などを登録せず、施設単位の実績と売上管理に目的を限定します。

GETTING STARTED

インストール後、すぐに始められます

専門的なデータベース設定やコマンド操作は不要。初回起動の案内に沿って、施設名と単価を登録します。

1

Windowsへインストール

セットアップファイルを実行し、デスクトップアプリとして導入します。

2

初期設定（4ステップ）

施設名と4区分の売上単価を入力。設定内容は後から変更できます。

3

目標を登録して運用開始

月間目標を設定し、施設ごとの人数入力から売上管理を始めます。

SUMMARY

売上管理を、
もっと分かりやすく。

訪看かんたん売上管理は、月次の人数入力から売上見込み・目標達成状況の確認までを、一つの分かりやすい画面にまとめます。

入力をシンプルに 計算を自動化 現在地を見える化

本資料の画面内数値・施設名は説明用サンプルです。実際の表示内容は登録データに応じて変わります。

本アプリでできること

  • 施設ごとの月次人数入力
  • 売上見込み・達成率の自動集計
  • 施設別・看護区分別の確認
  • 月締め・過去月の確認
  • CSV出力・バックアップ・復元
